# chatter

##new NIMChatter(options)
###options
初始化实例参数

| 参数 | 类型 |说明 | 
| --- | --- | --- |
| parent | string/HTMLElement| 插件注入节点
| appkey | String | appkey |
| account | String | 账号 |
| token | String | token |
| mode| String | simple:最小化集合的点对点聊天, full|
| to | String | simple模式下指定聊天对象，或者初始化后打开指定对象聊天面板|
